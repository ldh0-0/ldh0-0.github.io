  3. 递归处理左右两段

二分

数的范围

二分分为整数二分和实数二分
整数二分：

  • 如果有单调性可以二分，如果没有单调性也有可能可以二分，所以二分的本质并不是单调性
  • 本质：边界
  • 如：给定一个区间左半边满足右半边不满足二分就可以寻找这个性质的边界


步骤：

  1. 二分红边界点mid = (l+r+1)/2 找到中间值
    2. if(check(mid))如果是true(mid在红色区间中)答案就在[mid,r] 更新l=mid; 如果是false（mid在蓝色区间中）答案在[l,mid-1) 更新r=mid-1;
  2. 二分蓝边界点mid = (l+r)/2
    3. if(check(mid))如果是true(mid在蓝色区间中) 答案就在[l,mid] 更新r=mid; 如果是false(mid在红色区间中) 答案在[mid+1,r] l=mid+1;
注意： 
当l=mid时mid=(l+r+1)/2,l=mid+1时不变
因为当l=r-1时,mid向下取整,l=mid的结果就还是l=mid=l,所以要加1

高精度

高精度加法

给定一个数应该从个位开始存到高位
如：
数：123456789
数组：0 1 2 3 4 5 6 7 8
存 ： 9 8 7 6 5 4 3 2 1
这样产生进位只要在数组末尾补上一位，否则就将整个数组往右移一位。

前缀和与差分

前缀和

原数组：a1 a2 a3 … an
前缀和：si = a1 + a2 + … + an
如何求si s[i] = s[i-1] + ai

二维前缀和（子矩阵的和）

  1. s[i,j] 如何计算
    s[i,j] = s[i-1,j] + s[i,j-1] - s[i-1,j-1] + a[i,j]
  2. (x1,y1),(x2,y2)这一子矩阵中所有数的和如何计算
    s[x2,y2] - s[x2,y2] - s[x2,y1-1] + s[x1-1,y1-1]

    差分

    差分可以看成前缀和的逆运算

如给定一个数组a[n]=a[1],a[2]…,a[n];
在创建一个数组b[i]=b[1],b[2]…,b[i];
如果a[i]=b[1]+b[2]+…+b[i];
a数组就是b数组的前缀和 而b数组就是a数组的差分
如何得到b数组：

b[1]=a[1]-a[0];
b[2]=a[2]-a[1];
…
b[n]=a[n]-a[n-1];

如果使b[i]+l,就等于从a[i]开始每一个数加上l
如果使b[i]-l,就等于从a[i]开始每一个数减去l
从而可以想到只要让b[l]+c,b[r+1]-c就可以在O(1)的时间复杂度算出a[l]到a[r]的每个数加上c。

双指针算法

最长连续不重复子序列

转

  1. 遍历数组a中的每一个元素a[i], 对于每一个i，找到j使得双指针[j, i]维护的是以a[i]结尾的最长连续不重复子序列，长度为i - j + 1, 将这一长度与r的较大者更新给r。
  2. 对于每一个i，如何确定j的位置：由于[j, i - 1]是前一步得到的最长连续不重复子序列，所以如果[j, i]中有重复元素，一定是a[i]，因此右移j直到a[i]不重复为止（由于[j, i - 1]已经是前一步的最优解，此时j只可能右移以剔除重复元素a[i]，不可能左移增加元素，因此，j具有“单调性”、本题可用双指针降低复杂度）。
  3. 用数组s记录子序列a[j ~ i]中各元素出现次数，遍历过程中对于每一个i有四步操作：cin元素a[i] -> 将a[i]出现次数s[a[i]]加1 -> 若a[i]重复则右移j（s[a[j]]要减1） -> 确定j及更新当前长度i - j + 1给r。

位运算

二进制中1的个数

n的二进制表示中第k位是几
n = 15 = (1111)2

  1. 先把第k位移到最后一位 n>>k
  2. 看个位是几 x&1

n的第k位就为： n>>k&1

lowbit(x):返回x的最后一位1
例如：

  • x = 1010 lowbit(x) = 10
  • x = 101000 lowbit(x) = 1000

lowbit实现 x&-x

原因：
-x = ~x+1 （~是取反的意思）x的补码(反码+1)就是-x
得出公式： x&-x = x&(~x+1)
例如：
x = 1010…100..0
~x = 0101…011..1
~x+1 = 0101…100..0
x&(~x+1) = 0000…100…0
应用：可以统计数字中1的个数，每次将最后一个1减去，当数字减为0时，减去多少次就有多少个1。

区间合并

  1. 按区间左端点排序
  2. 扫描整个区间将有交集的进行合并,扫描过程中维护一个区间。当前区间碰到维护区间时有三种情况，1.是维护区间的子集2.和维护区间有交集3.和维护区间没有交集。第一种情况维护区间不变,第二种情况维护区间的右端点变为当前区间的右端点,第三种情况维护区间已经和后面的区间没有交集将维护区间拿出,再将维护区间更换为当前区间。

01背包问题

f[i][j] 表示只看前i个物品，总体积是j的情况下，总价值最大是多少。
result = max{f[n][0~V]}
f[i][j] :

  1. 不选第i个物品，f[i][j] = f[i-1][j];
  2. 选第i个物品，f[i][j] = f[i-1][j-v[i]]先将第i个物品的体积减去然后考虑第i-1物品的最大价值
    最后答案f[i][j] = max{1.2.}

f[0][0] = 0
重要的是思想将这种二选一的状态确定下来不断移动。
